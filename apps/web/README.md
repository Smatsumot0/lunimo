## 🧑‍💻 About

実用性とUXを重視して設計・開発しているアプリです。

React / Next.js をベースに、バックエンド（NestJS / Prisma）まで含めた
フルスタック構成で開発しています。

---

## 🎯 コンセプト

「毎日使える軽さ」と「直感的な操作性」を重視した、生理管理アプリ。

既存のアプリで感じた以下の課題を解決することを目的としています。

- 入力が面倒（datepicker・画面遷移が多い）
- 必要な情報にすぐアクセスできない
- UIが複雑で継続しづらい

---

## ✨ 特徴

- 7日間の横スクロールカレンダー
- 最小限の入力で記録できるUI設計
- 体調・服薬をワンタップで記録
- 必要な機能だけを選択できる柔軟な設計

---

## 🧩 主な機能

- 生理予定日の表示
- 生理開始日の記録
- 体調（好調 / 不調）の記録
- 痛みレベルの管理
- 服薬状況の記録（1日複数回対応）
- 生理周期の可視化
- 履歴一覧

## 🧠 設計のポイント

### UI/UX

- カレンダーは「1週間」に絞り、横スクロールで操作
- useEffectに依存しない設計で軽量化

### アーキテクチャ

- フロントエンドとバックエンドを完全分離
- APIはRESTベースで構築
- サーバーコンポーネントでのデータ取得を採用
